# CoolTech – AC Service Manager SaaS

CoolTech is a professional AC service management SaaS for technicians.

## 🚀 Features

- Phone OTP Authentication
- Customer Management
- Job Creation & Tracking
- Payment Status (Pending / Paid)
- Payment Method (Cash / UPI / Bank)
- Revenue Dashboard
- Multi-user Secure Isolation (Row Level Security)

## 🛠 Tech Stack

- Vite
- React
- TypeScript
- Supabase
- Tailwind CSS
- shadcn-ui
- React Query
